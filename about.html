About

MathRunner was created by Jordan Liu for CIT2011: Individual Assignment 2. I was required to build a Mathematics Practice Site for 9 to 12-year-olds. The user chooses which activity they would like to do: addition, multiplication, subtraction, or division exercises. This web site should randomly generate two (2) integer numbers and then ask the user to solve the problem based on the activity. This problem sets must be implemented using a level-based system, where the problems increase in difficulty as the user progresses through each problem set and level. Each level must contain at least 15 questions. A user is allowed to progress to the next level as long as they have achieved a pass rate greater than or equal to 80%.


This website was created using:

  • HTML5
  • CSS3
  • JavaScript
